Measure bar
5

1. Tap in the measure bar at any place between the start and the end of measure one. This will select all of the notes in both staffs.

2. Double tap in the same place. This will select all the measures in both staffs.

Discussion: The first method selects the notes individually while the second method selects whole measures. Selecting whole measures is a prelude to copy and paste operations.

3. Tap and drag between measure numbers directly on the bar line separator to expand or shrink the measure width (you can't shrink the measure width beyond a minimum default value). The measure number will turn yellow indicating the measure width has been manually modified.

4. Long press anywhere in the measure bar for the measure whose width you just changed to reset the measure to its default width.

5. Double tap on the staff name to select an entire staff.

Result: You have learned how to make more selections with the measure bar and drag and reset measure widths.
